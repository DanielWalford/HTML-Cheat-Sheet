HTML Cheat Sheet

Basic HTML Elements & Tags

Tag Name Description
<!DOCTYPE HTML> DOCTYPE First line of doc. Tells browser what version of HTML the document will be written in
<html> HTML Root element that defines the whole HTML document. All content goes between the HTML tags
<head> Head Contains information relating to document. Contains title, meta, script, link, styles etc
<title> Title Title of webpage. Viewed on tabs
<body> Body Defines the body of the document. Main content goes between these tags
<h1> h1-h6 Defines a heading, biggest to smallest.
<p> Paragraph Paragraphs go between these tags
<a> Anchor Creates links to a target URL. Can link pages within webpage or link to websites outside of webpage

HTML5 Elements & Tags

Tag Name Description
<header> Header Container for introductory content/nav links. Often contains headings, logos and authorship information
<nav> Nav Defines a block of navigational links
<main> Main Specifies main content of document. must NOT be a descendant of an article, aside, footer, header, or nav element
<footer> footer Defines footer of document. Often contains site map, contact information and copyright information
<section> Section Defines a section of a document to indicate a related grouping of semantic meaning
<article> Article Specifies independent, self-contained content.Should make sense on its own and it should be possible
to distribute it independently from the rest of the site
<aside> Aside Defines some content aside from the content it is placed in. The aside content should be indirectly
related to the surrounding content
<figure> Figure Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
Include src, alt and sizing attributes
<figcaption> Figcaption Defines a caption for a figure
